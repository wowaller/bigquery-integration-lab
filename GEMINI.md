# Gemini Workspace: Java & Python Tooling for BigQuery & Dataplex

This workspace contains enterprise tools for Google Cloud services, specifically BigQuery and Dataplex Data Lineage. It includes implementations in both Java and Python for:
-   **Schema Extraction**: Listing datasets, tables, and views with column-level schemas.
-   **Data Lineage Graph Visualization**: Finding upstream SQL jobs and source tables for a given target entity.
-   **Airflow/Composer Monitoring**: Python scripts to track Airflow DAG runs.

Check the `README.md` for detailed build and run instructions!
